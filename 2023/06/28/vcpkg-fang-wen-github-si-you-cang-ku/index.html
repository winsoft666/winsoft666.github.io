Vcpkg访问Github私有仓库

发表于 2023-06-28 更新于 2023-07-12 分类于 Vcpkg

1.在Github中添加TOKEN

2. 添加Windows环境变量：

VCPKG_KEEP_ENV_VARS值为GITHUB_TOKEN

GITHUB_TOKEN值为第一步生成的TOKEN

3.修改portfile.cmake

在vcpkg\ports目录下，找到对应库的portfile.cmake文件，加入AUTHORIZATION_TOKEN参数。
